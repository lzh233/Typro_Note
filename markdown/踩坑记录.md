## 1 conda环境中R语言 install.packages("hdf5r")报错

```shell
#报错信息
global.h:23:10: fatal error: hdf5.h: No such file or directory
#解决方法：conda环境中安装hdf5, 安装hdf5后直接安装hdf5r即可
conda install -c anaconda hdf5
install.packages("hdf5r")
```
